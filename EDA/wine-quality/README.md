# Wine Quality Dataset - EDA

## Objective
Perform exploratory data analysis (EDA) to understand the characteristics of the wine quality dataset.

## Dataset
Kaggle-Wine Quality dataset (WineQT.csv).  
The dataset is not included in this repository.
You can download it from:
https://www.kaggle.com/datasets/yasserh/wine-quality-dataset

## Analysis Includes
- Dataset overview and structure
- Missing value analysis
- Distribution of features
- Correlation analysis
- Key observations and insights

## Notes
This project focuses only on exploratory data analysis.
No machine learning or regression models are applied.
