# Titanic Dataset - EDA + Future Engineering 

## Project Overview

This project focuses on Exploratory Data Analysis (EDA) and Feature Engineering using the Kaggle Titanic dataset.
The goal is to understand the dataset deeply before building predictive models.

## Dataset
Kaggle-Titanic dataset (train.csv).
The dataset is not included in this repository.
You can download it from:
https://www.kaggle.com/competitions/titanic


## Purpose of This Project

-Practice data cleaning techniques

-Understand categorical encoding strategies

-Develop feature engineering intuition

-Prepare data for future modeling
